Black-box testing is a method of software testing that examines the functionality of an application without peering into its internal structures or workings.

White-box testing is a method of testing software that tests internal structures or workings of an application.

Poziomy testowania

  • Jednostkowe
  • Integracyjne
  • Systemowe

Jednostkowe

  • Czas wykonania wykonania: natychmiastowe
  • Automatyzacja: tak
  • Pokrycie: funkcja/procedura
  • Co testujemy?
    • Analiza ścieżek
    • Wartości brzegowe
    • Klasy równoważności

Integracyjne

  • Czas wykonania wykonania: krótkie
  • Automatyzacja: tak
  • Pokrycie: komunikacja międzymodułowa
  • Co testujemy?
    • Integracja między wartwami aplikacji
    • Integracja między modułami aplikacji
    • Integracja między systemami zewnętrznymi

Systemowe

Mockito

Mockowanie


                   import org.junit.Assert;
                   import org.junit.Test;
                   import org.mockito.Mockito;

                   import java.util.List;

                   public class MockitoTest {

                   @Test
                   public void shouldMockedListReturnSizeOf2() {
                   List list = Mockito.mock( List.class);
                   Mockito.when( list.size() ).thenReturn( 2 );
                   Assert.assertEquals( 2,list.size() );
                   }
                   }

Nagrywanie


                   import org.junit.Assert;

Rzucanie wyjątków


                   @Test( expected = IndexOutOfBoundsException.class )
                   public void shouldThrowPreviouslyRecordedException(){
                   List list = Mockito.mock( List.class );
                   Mockito .when( list.get( 9 ) )
                   .thenThrow( IndexOutOfBoundsException.class );
                   try {
                   list.get( 0 );
                   }catch( IndexOutOfBoundsException e ){
                   Assert.fail();
                   }
                   list.get( 9 );
                   }

Sprawdzanie wykonania


                   @Test
                   public void shouldVerifyIfTheMethodWasCalled()
                   {
                   Runnable runnable = Mockito.mock( Runnable.class );
                   runnable.run();
                   Mockito.verify( runnable, Mockito.times( 1 ) ).run();
                   }

Szpiegowanie obiektów


                   @Test
                   public void shouldVerifyIfTheMethodWasCalled()
                   {
                   List<String> list = new ArrayList<>();
                   List<String> spyList = Mockito.spy( list );

                   spyList.add( "test" );

                   Mockito.verify( spyList ).add( Mockito.anyString() );
                   }
