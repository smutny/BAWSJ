Code-smell

Bad smell in computer programming code, refers to any symptom in the source code of a program that possibly indicates a deeper problem.

Poziom aplikacji

  • Duplikacja kodu
  • Overengineering

Poziom klasy

  • Large class
  • Feature envy
  • Inappropriate intimacy
  • Refused bequest
  • Lazy class
  • Excessive use of literals
  • Cyclomatic Complexity
  • Downcasting
  • Orphan Variable or Constant class

Poziom metody

  • Too many parameters
  • Long method
  • Excessively long identifiers
  • Excessively short identifiers
  • Excessive return of data

Refaktoryzacja

Code refactoring is the process of restructuring existing computer code – changing the factoring – without changing its external behavior.

Zalety refaktoryzacji

  • Utrzymanie
  • Rozszerzalność
  • Modułowość
  • Czytelność

Wady refaktoryzacji

  • Koszt
  • Potencjalnie - nowe błędy

Przykładowe przekształcenia

  • Method rename
  • Variable/field rename
  • Add/Remove parameter
  • Inline Method
  • Introduce Builder
